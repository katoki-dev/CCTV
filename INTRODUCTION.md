# CEMSS - Campus Event management and Surveillance System

## Introduction

**CEMSS (Campus Event management and Surveillance System)** is an advanced AI-powered surveillance platform that combines real-time threat detection with intelligent conversational assistance. Built for modern security needs, CEMSS transforms traditional camera monitoring into a smart, proactive security solution.

## What is CEMSS?

CEMSS is a comprehensive surveillance system that leverages cutting-edge artificial intelligence to monitor, analyze, and respond to security events in real-time. Unlike conventional CCTV systems that require constant human supervision, CEMSS autonomously detects threats, analyzes visual data, and provides instant alerts—making security monitoring more efficient and reliable.

## Core Capabilities

### 🎯 Intelligent Detection

CEMSS employs multiple AI models to identify:

- **Person Detection**: Track individuals entering monitored areas
- **Fall Detection**: Immediate alerts for elderly care and workplace safety
- **Violence Detection**: Recognize aggressive behavior and confrontations
- **Crowd Detection**: Monitor crowd density and prevent overcrowding
- **Phone Detection**: Identify unauthorized mobile device usage in restricted areas
- **Motion Analysis**: Continuous background monitoring for unusual activity

### 🤖 AI Chatbot Assistant

A unique feature that sets CEMSS apart is its natural language assistant powered by Vision-Language Models (VLM). Users can:

- Ask questions about live camera feeds: *"What do you see on Camera 1?"*
- Query detection history: *"Show me recent falls detected today"*
- Check system status: *"How many cameras are currently active?"*
- Request visual analysis of specific events
- Interact using plain English—no technical expertise required

### 📊 Advanced Analytics

CEMSS provides comprehensive insights through:

- Interactive detection timelines
- Confidence score tracking for AI predictions
- Event distribution charts and statistics
- Exportable reports for compliance and record-keeping

### 🔔 Multi-Channel Alerts

Stay informed through multiple notification channels:

- **Email alerts** with detection snapshots and video clips
- **WhatsApp notifications** for instant mobile updates
- **In-app alerts** with severity-based filtering
- Complete alert history and audit trails

## Technology Highlights

CEMSS leverages state-of-the-art technologies:

- **YOLO (You Only Look Once)**: Industry-leading object detection with 85%+ accuracy
- **Ollama Framework**: Local AI inference ensuring data privacy (no cloud dependency)
- **Vision-Language Models**: MiniCPM-V and Moondream for visual understanding
- **Lightweight LLMs**: Qwen2.5:0.5b for fast, efficient conversational responses
- **Modern Web Stack**: Responsive Flask-based interface with real-time updates

## System Requirements

- **Operating System**: Windows, Linux, or macOS
- **Python**: Version 3.9 or higher
- **Memory**: Minimum 4GB RAM (8GB recommended)
- **Storage**: 10GB available space for models and recordings
- **Ollama**: Required for AI features

## Getting Started

CEMSS offers multiple deployment options:

1. **Quick Install**: One-click setup script for Windows (`CEMSS_Complete_Setup.bat`)
2. **Manual Installation**: Detailed guide in `INSTALLATION.md`
3. **Docker Deployment**: Containerized setup for production environments

Access the system at `http://localhost:5000` with default credentials (`admin/admin`).
